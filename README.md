## 기능목록

- [x] 체스말의 각 진영은 흑팀(알파벳 대문자), 백팀(알파벳 소문자)으로 구분한다.
- [x] 체스말은 위치를 가진다.
- [x] 게임시작시에 체스판이 초기화된다.
  - [x] 각 체스 말은 생성될때 default 위치를 가진다.
- [x] 체스판에서 말의 위치 값은 가로 위치는 왼쪽부터 a ~ h이고, 세로는 아래부터 위로 1 ~ 8로 구현한다.
